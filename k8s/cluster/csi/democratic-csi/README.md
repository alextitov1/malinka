# Democratic CSI (ZFS via NFS)

This directory contains the configuration for installing [Democratic CSI](https://github.com/democratic-csi/democratic-csi)
## Prerequisites

Ensure the ZFS host is fully configured. There is an Ansible playbook for this:
```bash
# Run this from the root of the workspace
ansible-playbook  node-init/services/zfs/zfs-Democratic-CSI-host-nfs.yaml
```
This playbook will:
1. Install and start the NFS server on the host.
2. Create the `storage/malinka-k8s` parent dataset.
3. Generate an SSH key (`playbooks/zfs/k8s-csi-key`) and add it to the host.

## Installation

**1. Add the Helm Repository**
```bash
helm repo add democratic-csi https://democratic-csi.github.io/charts/
helm repo update
```

**2. Create the namespace**
```bash
kubectl create namespace democratic-csi
```

**3. Configure your Values**
Create a `values.yaml` file (you can copy a template below). **Make sure you insert the private SSH key generated by Ansible:**
```bash
# cat playbooks/zfs/k8s-csi-key
```

**4. Install using Helm**
```bash
helm upgrade --install zfs-nfs democratic-csi/democratic-csi \
  --namespace democratic-csi \
  -f k8s/cluster/csi/democratic-csi/values.yaml
```

**5. Create a StorageClass**
```bash
kubectl apply -f k8s/cluster/csi/democratic-csi/storage-class.yaml
```
